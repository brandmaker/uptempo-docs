Save Summary table columns as presets

You can use Column Presets to save your customized table column settings in the Summary display mode. Saving column settings as a column preset lets you quickly and easily apply the same settings again at any time. This makes them a useful shortcut if you have different column configurations that you frequently need to switch between.

You can:

  • Save your current customized column configuration in the Summary display mode as a column preset at any time.

  • Select a previously created column preset to quickly apply its saved column configuration again.

  • Optionally, link column presets to views (saved display settings, like filters or color-coding) to also automatically apply a linked column preset when the view is selected.

Create column presets

After you customize the table column configuration in the Summary display mode, you can save the configuration as a column preset.

You can create as many column presets as you need. Whenever you select a column preset later, the column configuration saved to the preset is immediately applied again.

By default, column presets you create are exclusive to you, and aren't available to any other user in your Uptempo instance.

Administrators can additionally create column presets that are shared with all users in the organization, and manage those views. To learn more, see Manage Summary column presets for your organization.

Create a new column preset

  1. In the Activities section, set the Timeline/Summary toggle to the Summary setting.

  2. Click Configure Columns, and use the Configure Columns panel to customize the columns in the Summary table. After you modify the column configuration, a badge is displayed on the Column Preset Options button to indicate that you can save the column configuration as a column preset:

  3. Click Column Preset Options. The Preset Options panel opens.

  4. Type a name for your new preset into the Column Preset Name field.

  5. Optional: If you are an administrator, you can set the new preset to be available to all users in your Uptempo instance:

    • Click Only for me on the Preset Name field, then select For my organization instead.

  6. To finish creating your new column preset, click Save as new. The Preset Options panel closes, and your new view now appears in the Select Column Preset menu:

You have saved your custom column configuration as a column preset, and can now select the column preset at any time to apply the saved column configuration.

You can also create a new column preset by copying an existing preset. To do this, select the column preset you want to copy. Then, click ColumnPresetOptions_18x18.png Column Preset Options, enter a new name into the Preset Name field, and click Save as new.

The resulting column preset will have the same column configuration as the preset it was copied from, and you can use this as a base to make additional changes (see Edit column presets).

By default, column presets are displayed in the Select Column Preset menu in alphabetical order. If you are an administrator, you can customize the order in which organizational column presets are displayed in the menu. For instructions, see Manage Summary column presets for your organization

    Copy column presets

    You can also create a new column preset by copying an existing column preset.

    Copy an existing column preset

    1. Apply the column preset you want to copy (see Apply column presets).

    2. Click Column Preset Options and enter a new name into the Column Preset Name field.

    3. To finish copying the column preset, click Save as new.

    The resulting column preset will have the same column configuration as the column preset it was copied from, and you can use this as a base to make additional changes (see Edit column presets).

      Apply column presets

      After you have created a column preset, you can quickly recall the saved column configuration in the Summary display mode by applying the preset.

      Apply an existing column preset

      1. In the Activities section, click the Select Column Preset menu to see available column presets. Column presets are displayed in two sections:

        My Presets

        Lists personal column presets that you have created yourself. If you have not created any column presets yet, this section is hidden.

        Organization Presets

        Lists column presets created by administrators for all users in your Uptempo instance. This section is always visible.

        There is no default column preset. If there are no personal or organization presets available yet, the Select Column Preset menu is empty.

      2. Click on the column preset you want to use to apply it.

      The column configuration saved in the preset is immediately applied to the Summary table.

      The system will remember the last column preset you applied, and will automatically apply it whenever you return to the Summary display mode. This includes after you have navigated to a different section, closed your browser, or signed out of Uptempo.

        Edit column presets

        After you have created a column preset, you can make changes to it at any time, such as changing its name and saved column configuration.

        If you are an administrator, you can also convert an Only for me column preset into a For my organization column preset (and vice versa).

        Rename a column preset

        1. In the Summary display mode, use the Select Column Preset menu to select the column preset you want to rename.

        2. Click Column Preset Options. The Column preset options panel opens.

        3. Type the new name for the column preset into the Column Preset Name field.

        4. To apply your changes, click Save changes.

          If you click Save as new, this will make a copy of the column under the new name, and will not change the name of the original view.

        You have renamed the column preset, and it will be displayed in the Select Column Preset menu under its new name.

          Change a column preset's saved column configuration

          1. In the Summary display mode, use the Select Column Preset menu to apply the column preset you want to modify.

          2. Click Configure Columns, and use the Configure Columns panel to customize the columns in the Summary table. After you modify the saved column configuration, a badge is displayed on the Column Preset Options button to indicate that there are unsaved changes.

          3. Click Column Preset Options. The Column preset options panel opens.

          4. To save the new column configuraton to the selected column configuration, click Save changes.

          You have changed the column preset's saved column configuration, and the new column configuration will be used whenever you apply the column preset from now on.

            Change a column preset's type (for administrators)

            As an administrator, you can change a column preset's type from Only for me to For my organization (and vice versa).

            1. In the Summary display mode, use the Select Column Preset menu to select the column preset for which you want to change the type.

            2. Click Column Preset Options. The Column preset options panel opens.

            3. Click the currently set column preset type ( Only for me or For my organization) beside the Column Preset Name field. Click the view type that is not currently selected to change to it.

            4. To apply your changes, click Save changes.

            If you also change the column preset's name and click Save as new, this will make a copy of the column preset under the new name and with the new type, and will not change the name or type of the original column preset.

              Delete column presets

              If you no longer need a column preset, you can delete it at any time.

              Delete a column preset

              1. In the Summary display mode, use the Select Column Preset menu to select the column preset you want to delete.

                All users can delete their own column presets, listed under My Column Presets. Only administrators can delete views listed under Organization Presets, including column presets that were created by other administrator users.

              2. Click Column Preset Options. The Column preset options panel opens.

              3. Click Delete preset.

              4. To permanently delete the column preset, click Delete in the confirmation dialog.

              The Summary table reloads with the next available column preset applied (if any), and the deleted preset no longer appears in the Select Column Preset menu.

                Link column presets to views

                If you frequently use a column preset with a particular view, you can link the preset to the view. When a column preset is linked to a view, selecting the view also automatically applies the column preset at the same time.

                In the Select View menu, views that have a linked column preset are marked with the Linked column preset icon:

                Link a column preset to a view

                1. In the Activities section, use the Select View menu to select the view you want to link to a column preset.

                  • You can do this in either the Timeline or Summary display modes.

                2. Click View Options. The View Options panel opens.

                3. Click Link a column preset to this view. The Linked column preset menu is displayed.

                4. Use the Linked column preset menu to choose the column preset you want to link to the selected view.

                  If you are not an administrator, you can only link column presets to your personal views (My Views).

                5. To finish linking the column preset to the view, click Save changes.

                The column preset is linked to the view. When you select this view, the linked column preset will also be applied automatically.

                When a column preset is linked to a view, the linking only works in one direction: selecting the view automatically applies the column preset, but selecting the column preset does not automatically select the linked view.

                  Unlink or change the linked column preset on a view

                  1. In the Activities section, use the Select View menu to select the view on which you want to unlink or change the column preset.

                    • You can do this in either the Timeline or Summary display modes.

                  2. Click View Options. The View Options panel opens.

                  3. Click Link a column preset to this view. The Linked column preset menu is displayed:

                    • To change the linked column preset: Use the Linked column preset menu to select the new column preset you want to link to the view.

                    • To unlink the currently linked column preset. Click Remove on the current selection in the Link view to column preset menu.

                    Only administrators can unlink or change linked column presets on Organization Views.

                  4. To apply your changes, click Save changes.

                  You have unlinked or changed the column preset on a view. Your changes take effect immediately.
